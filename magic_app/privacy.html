🔒 Политика конфиденциальности

Последнее обновление: 22 мая 2026 г.

Мы, команда «Герой Сказок», уважаем вашу конфиденциальность и заботимся о безопасности данных ваших детей.

1. Какие данные мы собираем

  • Имя и возраст ребёнка — для персонализации общения с Люциком.
  • Электронная почта родителя — для авторизации и восстановления доступа.
  • История разговоров — для улучшения работы ИИ и отслеживания прогресса.
  • Статистика страхов и достижений — для родительского контроля.
  • Технические данные (тип устройства, браузер) — для оптимизации приложения.

2. Как мы используем данные

  • Персонализация ответов волшебного кота Люцика.
  • Формирование статистики для родительского кабинета.
  • Улучшение алгоритмов ИИ и качества сказкотерапии.
  • Обеспечение безопасности и предотвращение злоупотреблений.

3. Использование сторонних сервисов

Наше приложение использует следующие сторонние сервисы:

  • DeepSeek API — для генерации ответов и сказок. Ваши сообщения обрабатываются ИИ DeepSeek в обезличенном виде.
  • Яндекс SpeechKit — для озвучивания голосов персонажей. Текст сообщений передаётся в Яндекс для синтеза речи.
  • Neon (PostgreSQL) — для хранения данных в зашифрованной базе данных.
  • Vercel — для хостинга и выполнения серверных функций.

Все сторонние сервисы соответствуют стандартам безопасности и не используют ваши данные в своих целях.

4. Хранение и защита данных

Ваши данные хранятся на защищённых серверах в Европе. Пароли хешируются. Доступ к базе данных ограничен. Мы регулярно обновляем систему безопасности.

5. Права родителей

  • Вы можете запросить удаление всех данных вашего ребёнка, написав нам на support@geroy-skazki.ru.
  • Вы можете экспортировать историю разговоров и статистику из родительского кабинета.
  • Вы можете отключить голосовые функции в любой момент.

6. Возрастные ограничения

Приложение предназначено для детей от 3 до 12 лет под контролем родителей. Регистрация осуществляется только родителями.

7. Изменения политики

Мы будем уведомлять вас об изменениях в политике конфиденциальности через родительский кабинет и по электронной почте.

8. Контакты

По всем вопросам: support@geroy-skazki.ru
