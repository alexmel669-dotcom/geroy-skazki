# 🐱 Люцик — Герой Сказок

ИИ-помощник для преодоления детских страхов через игры и сказки.

## 🚀 Технологии

- **Frontend:** Vanilla JS (ES Modules), PWA
- **AI:** DeepSeek Chat API
- **TTS:** Яндекс SpeechKit
- **STT:** Web Speech API
- **Backend:** Vercel Serverless Functions
- **Auth:** JWT

## 📦 Установка

```bash
# Клонирование
git clone https://github.com/your/geroy-skazki.git
cd geroy-skazki

# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env файл

# Запуск в development
npm run dev

# Деплой
npm run deploy
